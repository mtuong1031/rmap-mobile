# RMap

RMap is an Android application designed to help IT students and developers orient their skills and build personalized learning roadmaps based on career goals[cite: 5].
## Getting started

### Prerequisites
* Android Studio (Ladybug or newer recommended)
* JDK 17+
* Android SDK API 24+ (Android 7.0)

### Setup and Run
1. Clone the repository:
   ```
   git clone https://github.com/your-team/rmap-mobile.git 
    ```
2. Open and Build:
   Open the project in Android Studio and wait for Gradle sync to complete.

3. Run:
   Press Shift + F10 or the Run icon to deploy the app to an emulator or physical device.

## Project structure

* app/ 
    * ui/ — Compose screens & ViewModels.
    * data/ — API services & Repository implementations
    * domain/ — Business logic & models.
    * core/ — Constants, utilities & network config.
* docs/ — Documentation (SRS, specs)
